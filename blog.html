Effective Use of VS Code

Alright, this may seem like a dumb post, but I never took the time to really learn some shortcuts with VS Code before starting my software engineering job. Doing these will not only save you a lot of time coding, but will also make pair programming less awkward and time-consuming, especially if you’re like me and the only other person on your tech team is your supervisor.

Here are some of the commands I’ve found the most useful. Feel free to review more by navigating to: File > Preferences > Keyboard Shortcuts.

These shortcuts are for Mac (but probably similar for Windows too!):
Cmd + T: to search for a file within huge code bases
Cmd + D: selects the next instance of the selection (additionally you can update the selection all at once!)
Cmd + F: to search for a word in the current file
Cmd + Shift + F: to search a word in all the files in code base
Cmd + L: to select current line
Shift + Alt + up/down arrow: copy line above/below
Option + Up: move line up
Option + Down: move line down
Cmd + N: Creates a new file
Cmd + O: Open file prompt
Cmd + S: Saves current file

Additionally, here’s some really good extensions I’ve started using that have been super helpful:
HTML boilerplate: if you use HTML at all, get this extension. It creates all those forgetful HTML pieces when you create a new HTML file.
SQLTools: A SQL formatter. I use SQL at work and this is a great tool to format your file. Install this and when you write some SQL or use VS Code to visually edit your SQL code. On save the file should format to provide visually appealing SQL code.
PHP Intelephense: This is another formatter that’s great as I use PHP at work as well. It provides code completion, code folding (Premium edition), and Detailed hover with links to official PHP documentation to name a few.

Finally, another helpful tip is the ‘Outline’ tool at the bottom left of the VS Code window. This is really handy if you’re trying to wade through a file with large amounts of code. This tool will help guide you to where variables and functions are defined within the file. The tree of items can also be sorted by position within the file they are defined, name, or type.
